A simple shell program that simulates the behavior of a unix
command line interpreter.

It's a simple shell beacause it only incorporate few basic
functionalities of a standard shell.

The main.h file contains all function prototypes used in the 
program; the main.c file contains the main function and serve as 
an entry point for all functions used. Other .c files are basically 
files that contain the function definitons.

The AUTHORS file display the contributor(s) to th project.
